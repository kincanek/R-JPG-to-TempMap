# Conversión de Imágenes Térmicas DJI R-JPG a TIF

Este software se ha diseñado específicamente para transformar imágenes térmicas R-JPG, capturadas con el dron DJI-M3T, a formato TIF. El objetivo principal es permitir la creación de ortomosaicos utilizando el software Agisoft Metashape.

Además de la conversión, el programa también se encarga de exportar los metadatos asociados a cada imagen.

## Referencias y Recursos
DJI Thermal SDK: Herramientas y recursos para trabajar con imágenes térmicas de DJI. Puedes obtener más información y descargar el SDK [aquí](https://www.dji.com/downloads/softwares/dji-thermal-sdk).

FLIR/DJI IR Camera Data Parser (Versión Python): Una herramienta útil para el análisis y procesamiento de imágenes térmicas. El repositorio está disponible en [GitHub](https://github.com/SanNianYiSi/thermal_parser)  .
